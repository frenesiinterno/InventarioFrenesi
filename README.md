# InventarioFrenesi

Proyecto de gestión de inventario.

## Instalación

1. Instalar dependencias del backend:

```
npm install
```

2. Instalar dependencias del cliente:

```
cd client
npm install
```

## Ejecutar

- Backend: `node server.js` o `npm start` en la raíz
- Cliente: `npm start` dentro de `client`

## Notas
- Añadir archivo `.env` con la configuración de base de datos y variables sensibles.
